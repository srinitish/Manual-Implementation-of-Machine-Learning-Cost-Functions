# Manual Implementation of Machine Learning Loss Functions in Python

## Overview
This project demonstrates the **manual implementation of common machine learning cost functions** using pure Python without relying on deep learning libraries.  
The goal of this project is to understand the **mathematical foundation of loss functions** used in regression and classification models.

These implementations show how machine learning frameworks internally compute loss during model training.

---

## Implemented Loss Functions

### Regression Loss Functions
- Mean Squared Error (MSE)
- Mean Absolute Error (MAE)

### Classification Loss Functions
- Binary Cross Entropy
- Categorical Cross Entropy
- Sparse Categorical Cross Entropy
